# calculates the area under the curve for the given model

This function calculates the Area Under the Curve value for the given
model. It uses the
[`pROC::auc()`](https://rdrr.io/pkg/pROC/man/auc.html) function from the
`pROC` package.

## Usage

``` r
calculate_auc(model, data, .progress = FALSE)
```

## Arguments

- model:

  A model generated by the function
  [`mv_model_cox()`](https://metodologianavarrabiomed.github.io/mice-ext-val/reference/mv_model_cox.md)
  or
  [`mv_model_logreg()`](https://metodologianavarrabiomed.github.io/mice-ext-val/reference/mv_model_logreg.md)

- data:

  Multiple imputed data organized as a long dataset

- .progress:

  `TRUE` to render the progress bar `FALSE` otherwise

## Value

The same `model` passed as a parameter with the Harrell C Index value
stored in `$auc`

## Examples

``` r
if (FALSE) { # \dontrun{
calculate_auc(model, data)
} # }
```
